# Customer Segmentation Project

## Objective

To segment customers based on their annual income and spending behavior using K-Means Clustering.

## Tools Used

* Python
* Pandas
* Scikit-Learn
* Matplotlib
* Jupyter Notebook

## Dataset

Mall Customers Dataset

## Process

1. Loaded customer dataset
2. Selected Annual Income and Spending Score features
3. Applied K-Means Clustering
4. Visualized customer groups using scatter plot
5. Saved clustered customer data

## Output Files

* customer_segments.csv
* customer_segmentation.png

## Outcome

Customers were grouped into different segments based on spending patterns and income levels, helping businesses understand customer behavior for targeted marketing.
